AlkeWallet - Sistema de Gestión de Transferencias autora Alejandra Duarte para bootcamp Talento Digital

AlkeWallet es un proyecto de base de datos relacional desarrollado en PostgreSQL que simula el funcionamiento básico de una billetera digital. Permite gestionar usuarios, monedas y transacciones, incluyendo conversiones a moneda local (CLP).
Este proyecto es de uso educativo.
