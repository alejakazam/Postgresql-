# AlkeWallet - Sistema de Gestión de Transferencias

Proyecto desarrollado por Alejandra Duarte para el bootcamp Talento Digital.

AlkeWallet es un proyecto de base de datos relacional desarrollado en PostgreSQL que simula el funcionamiento básico de una billetera digital.
Permite gestionar usuarios, monedas y transacciones, incluyendo conversiones a moneda local (CLP).

---

## I. Tecnologías utilizadas

* PostgreSQL
* SQL

---

## II. Descripción del sistema

El sistema permite:

* Gestión de usuarios
* Administración de monedas
* Registro de transacciones
* Conversión de monedas a CLP
* Modelado de base de datos relacional

---

## III. Características principales

* Diseño de base de datos estructurado
* Uso de claves primarias y foráneas
* Integridad de datos mediante restricciones
* Simulación de operaciones financieras básicas
